# Sasu Blockchain

Sasu adalah blockchain baru berbasis Proof of Work (PoW) dengan total supply 50 juta token.

## Fitur
- Mining berbasis PoW
- Wallet untuk menyimpan token
- API node menggunakan Flask

## Cara Menjalankan

### 1. Instalasi Dependensi
```bash
pip install -r requirements.txt
```

### 2. Node
```
python node.py
```
### 3. mining
```
python miner.py
```
